# SlotGo

讓空檔變現的打工平台

## Introduction
SlotGo 是一個讓空檔變現的打工平台，旨在幫助用戶能快速找到短期工作，並讓企業或雇主在面臨突發性人力短缺時，可以有一個能快速補上短缺人力的管道。

## Project Repository
* 前端：https://github.com/Kayxue/WorkingSystemFrontend
* 後端：https://github.com/Kayxue/WorkingSystemBackend
* App：本專案

## Features
- 簡化的申請與審核流程：雇主能基於應聘者的基本資料與歷史評價快速做出決策。
- 應聘者工作排程管理機制：主動提示或禁止時間衝突的申請，降低應聘者因疏忽造成的排程錯誤
- 雙向驗證的打卡系統：應聘者輸入驗證碼，雇主確認記錄，確保出勤的真實性，減少勞資爭議
- 雙方互評機制：為媒合的可靠性提供數據參考
- 應聘者與雇主間的即時私訊管道：方便雙方協調工作細節或詢問相關資訊。
